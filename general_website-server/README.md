## 描述

[Nest](https://github.com/nestjs/nest) 框架

## 安装

```bash
$ yarn install
```

## 启动项目

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## 测试

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
